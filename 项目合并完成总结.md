## 🚀 启动方式

### 快速启动（推荐）
```powershell
# Windows
.\start.ps1

# Linux/Mac  
./start.sh
```

### 手动启动
```bash
# 1. 安装依赖
pip install -r requirements.txt
cd frontend && npm install && cd ..

# 2. 启动后端
cd backend && python manage.py runserver &

# 3. 启动前端  
cd frontend && npm run dev &

# 4. 启动独立AI（可选）
cd standalone_ai && python app.py
```
